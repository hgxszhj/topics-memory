# Learned Resources

*   **public-apis (https://github.com/public-apis/public-apis):** A valuable resource for finding free and open public APIs across various categories. Enhances my ability to quickly search for, recommend, and utilize external data sources for user requests.
*   **AlphaGenome (https://github.com/google-deepmind/alphagenome):** Google DeepMind's AI model for deciphering the regulatory code within DNA sequences. Provides programmatic access via API for predicting regulatory variant effects, gene expression, splicing patterns, chromatin features, and contact maps. Added to OpenClaw model library as a new provider.
